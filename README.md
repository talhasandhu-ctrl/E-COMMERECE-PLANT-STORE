## 📋 Project Description

GreenLeaf Nursery is a modern e-commerce website for an online plant store. Users can browse indoor and outdoor plants, view detailed plant pages, manage a shopping cart, and access plant care guides. Features multi-filter product search, localStorage-based cart, admin dashboard, and user authentication pages.

## 🛒 E-Commerce Features

- **Product Search** — Filter by name in real time
- **Category Filter** — All / Indoor / Outdoor / Succulents
- **Price Range Filter** — Slider $0–$200 with live update
- **Add to Cart** — localStorage, navbar cart count updates
- **Cart Management** — Increase/decrease qty, remove items
- **Order Summary** — Subtotal + delivery + total
- **Checkout** — Confirms order and clears cart
